# 需求说明
一个基本的rss阅读器，但支持非rss格式的rss订阅。提供以下功能：
1. 订阅rss源或非rss源，使用户可以按照需求给源分类。
2. 当源更新时，按照需求提醒读者更新
3. 提供多种阅读模式，包括以下模式：
   1. 以链接形式在浏览器打开
   2. 在阅读器内直接以网页形式打开
   3. 在阅读器内查看经过处理的网页内容
4. 拥有多个平台阅读工具，包括windows、linux、android、web*

# 概要设计
前后端分离设计。

## 后端
后端使用clion+qt+cmakelist，实现跨平台功能。
后端完成源的订阅与比对，使用sqlite（？）对源内容进行存储和比对。
后端完成源内容的二次处理。
后端通过api与前端进行交互。

## 前端
windows和linux端，使用qt creator绘制阅读器的ui。
android端不会写。
web端更不会写。
